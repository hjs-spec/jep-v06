# JEP v0.6 Draft Set

This package contains three draft documents:

1. `draft-wang-jep-judgment-event-protocol-06.md`  
   JEP-Core -06: neutral verifiable event protocol for judgment acts.

2. `draft-wang-jep-profiles-00.md`  
   Optional trust profiles and interoperability bindings.

3. `draft-wang-jep-conformance-00.md`  
   Schemas, validation result model, test vectors, and conformance behavior.

This set is designed as a v0.6 release package:
- Core is stable and narrow.
- Profiles are optional and extensible.
- Conformance can evolve quickly without destabilizing Core.
